# programa-avisoIp-Vb.Net
Programa creado hace unos años para poder informar a un negocio del cambio de Ip en un servidor con ip dinamica para informar de la nueva
y poder conectarse remotamente para subir backups de bsases de datos, este programa lo desarrolle por voluntad propia para poder conectarnos al servidor donde guardabamos 
los backups sin tener que estar llamando a la oficina para que nos informara de la nueva ip.
